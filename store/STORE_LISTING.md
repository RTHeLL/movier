# Материалы для Chrome Web Store — Movier

## Название

Movier

## Краткое описание (≤ 132 символа)

Плееры на Кинопоиске: смотрите фильмы и сериалы прямо на странице и без лишней рекламы в плеерах.

(97 символов)

## Полное описание

Movier добавляет на страницы фильмов и сериалов Кинопоиска удобную кнопку просмотра и открывает онлайн-плееры, не уходя с карточки тайтла. В плеерах дополнительно режется реклама — и по сети, и в самих фреймах.

Что умеет расширение

• Кнопка «Смотреть бесплатно!» на kinopoisk.ru/film и kinopoisk.ru/series
• Несколько источников воспроизведения во вкладках, если сервис вернул больше одного плеера
• Блокировка рекламных запросов (declarativeNetRequest) и скрытие типичных рекламных блоков во фреймах плееров
• Автообновление правил фильтров с kurduk.store
• Быстрый возврат с hd.kinopoisk.ru на обычную карточку www.kinopoisk.ru

Как пользоваться

1. Установите Movier и примите условия в настройках расширения.
2. Откройте любой фильм или сериал на www.kinopoisk.ru.
3. Нажмите «Смотреть бесплатно!» и выберите нужный плеер.

Для кого

Для тех, кто ищет фильмы и сериалы на Кинопоиске и хочет сразу перейти к просмотру в браузере — с меньшим количеством рекламы в плеерах.

Конфиденциальность и данные

Список плееров предоставляет сервис kp.apiget.ru. Расширение передаёт только ID фильма/сериала и технические идентификаторы. Полный HTML страницы и cookie Кинопоиска не отправляются.

Политика конфиденциальности: https://kurduk.store/privacy.html
Сайт и установка: https://kurduk.store/
Инструкция: https://kurduk.store/#/install

Movier не связан с Яндексом или Кинопоиском официально.

## Категория

**Для удовольствия** (Fun)

Почему эта категория: основная ценность — просмотр фильмов и сериалов (развлечение), а не продуктивность, покупки или инструменты разработчика. Так же размечены похожие расширения в Store.

Альтернатива (хуже подходит): «Инструменты» — только если акцент сместить на блокировку рекламы без функции плееров.

## Single purpose (поле в Dashboard)

Онлайн-плееры на страницах фильмов и сериалов Кинопоиска и блокировка рекламы в этих плеерах.

## Permission justifications

### storage
Хранение технических ID, флага согласия на политику конфиденциальности и URL сервера фильтров рекламы.

### alarms
Периодическое обновление remote DNR-правил блокировки рекламы (раз в 6 часов).

### declarativeNetRequest
Блокировка рекламных доменов и трекеров в сетевых запросах плееров по статическим и обновляемым правилам.

### Host permission: https://kp.apiget.ru/*
Запрос списка iframe-плееров по ID фильма/сериала (основная функция расширения).

### Host permission: https://kurduk.store/*
Загрузка JSON-правил фильтров рекламы и открытие политики конфиденциальности.

### Host permission: https://rthell.github.io/*
Запасной источник фильтров при публикации через GitHub Pages.

### Content scripts на kinopoisk.ru
Встраивание кнопки и UI плееров только на страницах фильмов/сериалов.

### Content scripts на http(s)://*/* (all_frames)
Element-hiding рекламы во фреймах плееров (плееры загружаются с разных доменов). Не используется для сбора данных со всех сайтов.

## Remote code / remote data

Удалённый JavaScript не выполняется. Загружаются:
• JSON конфигурация DNR-правил;
• URL iframe плееров (изолированный контекст iframe, sandbox).

## Скриншоты (подготовить 3–5 шт., 1280×800)

1. Страница фильма на Кинопоиске с кнопкой «Смотреть бесплатно!» в шапке.
2. Открытая панель с вкладками плееров и воспроизведением.
3. Экран согласия / политика перед первой загрузкой.
4. Страница настроек расширения (согласие + filtersBaseUrl).
5. (опционально) Popup расширения с подсказкой.
