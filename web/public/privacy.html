Политика конфиденциальности

Расширение Movier и сайт kurduk.store. Дата обновления: 19 июля 2026 г.

1. Оператор

Политика описывает обработку данных расширением Movier для Chromium и связанным сайтом https://kurduk.store/.

2. Какие данные обрабатываются

Расширение может обрабатывать следующие данные:

  • ID фильма или сериала — извлекается из URL страницы Кинопоиска (kinopoisk.ru/film/… или /series/…) и передаётся сервису плееров для получения списка iframe.
  • Технические идентификаторы расширения — случайно сгенерированные значения UID, apiget_uid и fake_yandex_login, хранимые в chrome.storage.local. Они нужны для совместимости с API плееров и не являются вашим логином или паролем Яндекса/Кинопоиска.
  • Флаг согласия (privacyAccepted) и настройка URL обновления фильтров рекламы (filtersBaseUrl).
  • Правила блокировки рекламы — загружаются с разрешённого сервера фильтров (по умолчанию https://kurduk.store/filters/movier-dnr.json) и применяются через declarativeNetRequest.

Расширение не отправляет полный HTML страницы Кинопоиска, cookie сайта Кинопоиска, пароли, платежные данные и историю посещений произвольных сайтов на наши серверы.

3. Куда передаются данные

  • kp.apiget.ru — сторонний сервис списка плееров. Передаются ID фильма, технические идентификаторы и минимальный stub-HTML (не содержимое вашей сессии на Кинопоиске).
  • kurduk.store (или другой разрешённый filtersBaseUrl) — загрузка JSON-правил блокировки рекламы.

Обработка на стороне kp.apiget.ru регулируется политикой этого сервиса. Мы не контролируем хранение и дальнейшее использование данных этим оператором.

4. Цели обработки

  • показ плееров на страницах фильмов/сериалов Кинопоиска;
  • блокировка рекламных сетевых запросов и элементов в плеерах;
  • обновление списков блокировки;
  • учёт согласия пользователя перед запросом плееров.

5. Хранение

Идентификаторы и настройки хранятся локально в chrome.storage.local на вашем устройстве. Вы можете удалить их, удалив расширение или очистив данные расширения в настройках Chrome.

6. Передача третьим лицам и продажа

Мы не продаём персональные данные. Передача на kp.apiget.ru необходима для основной функции расширения (получение плееров). Данные не используются для кредитного скоринга.

7. Согласие

Перед первой загрузкой плееров расширение запрашивает явное согласие в интерфейсе на странице Кинопоиска и/или на странице настроек расширения. Без согласия запрос к сервису плееров не выполняется.

8. Дети

Расширение не предназначено для детей младше 13 лет и не собирает данные детей осознанно.

9. Изменения

При существенном изменении практик обработки данных политика будет обновлена на этой странице, а в расширении может потребоваться повторное согласие.

10. Соответствие Chrome Web Store User Data Policy

The use of information received from Google APIs (если применимо) and user data handled by this extension will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

11. Контакты

Вопросы по конфиденциальности: через страницу kurduk.store или issues репозитория проекта Movier на GitHub.

Сайт Movier (поиск фильмов) может отдельно обращаться к API Кинопоиска / вашему backend; это описано в настройках развёртывания и не смешивается с данными расширения без необходимости.
